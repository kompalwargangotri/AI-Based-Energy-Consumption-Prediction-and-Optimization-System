# AI-Based Energy Consumption Prediction and Optimization System

# Description
An AI-based system that predicts and optimizes energy consumption using Machine Learning and Deep Learning techniques. The project analyzes historical energy data to forecast future usage and provides smart recommendations to reduce energy wastage and improve efficiency.

---

#Objectives
- Predict future energy consumption
- Analyze historical energy usage patterns
- Reduce energy wastage
- Minimize electricity cost
- Improve overall energy efficiency

---

# Technologies Used
- Python
- NumPy, Pandas (Data Processing)
- Scikit-learn (Machine Learning)
- TensorFlow, Keras (Deep Learning)
- Matplotlib (Visualization)
- Jupyter Notebook / Google Colab
- Streamlit (Web Application)

---

# Dataset
- Time-series energy consumption data
- Features include:
  - Date & Time
  - Energy usage
  - Temperature
  - Heating/Cooling indicators
  - Solar radiation
  - Holiday data

---

# Models Used
- Random Forest
- Support Vector Machine (SVM)
- Decision Tree
- Ridge Regression
- LSTM (Deep Learning)

---

# Results
- SVM achieved the highest accuracy (~83.7%)
- The system successfully predicts energy consumption
- Helps in reducing energy wastage and electricity cost

# How to Run (Jupyter/Colab)

1. Clone the repository:
   git clone [https://github.com/your-username/ai-energy-consumption-prediction.git](https://github.com/kompalwargangotri/AI-Based-Energy-Consumption-Prediction-and-Optimization-System/blob/main/APPLIED_AI_FINAL_PROJECT.ipynb)

2. Navigate to the project folder:
   cd AI-Based Energy Consumption Prediction and Optimization System

3. Install required libraries:
   pip install numpy pandas scikit-learn tensorflow matplotlib

4. Open the notebook:
   jupyter notebook

5. Run the `.ipynb` file step by step

---

# Run Using Streamlit (Web App)

1. Install Streamlit:
   pip install streamlit

2. Create a file named `app.py` and add your model code

3. Run the Streamlit app:
   streamlit run app.py

4. Open in browser:
   http://localhost:8501
